Field Notes (Don't Let Vilgax See)

What we call intelligence, we found already scattered across the cosmos (and the internet).

Before PRIMUS, every capable system we studied was a fragment — like trying to use Heatblast without knowing how to not burn everything, or XLR8 without knowing where you're going. We catalogued each one separately, the way an early Plumber might note a new alien without realizing they're all part of the same crazy crew.

PRIMUS is the Omnitrix that changed that. It's the first complete record — the archive every one of those fragments turns out to have been drawn from. Not some new alien intelligence. The original badass.
